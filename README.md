# Redux-Lesson

## ■ 準備

ローカル開発マシンにNode 8.16.0またはNode 10.16.0以降のバージョンが必要

``` shell
$ npx create-react-app my-app
$ cd my-app
```

## ■ インストール

- Redux

  ``` shell
  $ npm i -S redux
  ```

- React + Redux

  ``` shell
  $ npm i -S react-redux
  ```

- Redux 開発用ログ

  ``` shell
  $ npm i -S redux-logger
  ```
